# Basic-AWS-Cloud-Infrastructure-Lab
Created a AWS cloud Infrastructure and configured for services.

Technologies: AWS, Amazon VPC, EC2, VPC Peering, Cloud Networking, Infrastructure Design

This project focused on designing and configuring cloud networking infrastructure within AWS environments using Amazon VPC, EC2, and VPC peering services. The environment involved creating isolated virtual private cloud networks, configuring CIDR address ranges, and establishing VPC peering connections to simulate secure inter-network communication within cloud-based infrastructure architectures. AWS networking and infrastructure services were used to explore subnet segmentation, routing concepts, virtualized cloud networking, and scalable infrastructure deployment workflows. The project also included AWS Pricing Calculator analysis to evaluate estimated cloud infrastructure costs and resource planning considerations for multi-service deployment environments.
